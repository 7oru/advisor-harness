# Post-Run Review Policy

The advisor reviews completed runs for:

- missed advisor opportunities
- unnecessary advisor calls
- bad memory proposals
- risky or unsupported claims
- suggested routing policy improvements
- proposed memory schema, executor prompt, and advisor prompt improvements

Policy and improvement proposals are advisory only. The harness must not auto-edit policy, schema, or prompt files. Every structured improvement proposal must remain in `status: proposed` and require human approval before any later apply path can use it.
