# Security Questionnaire Workflow

This scaffold uses the generic advisor harness for security questionnaire tasks.

Expected run artifacts:

- `answers_draft.md`
- `evidence_links.md`
- `risk_flags.md`
- `open_questions.md`
- `memory_proposals.jsonl`
- `advisor_reviews.jsonl`

MVP behavior is intentionally conservative: all customer-visible answers require human confirmation.
